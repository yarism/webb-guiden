Snabba tips: Ifall du använder Internet Explorer så finns det en chans att en del webbsidor inte riktigt fungerar som de ska. Testa att byta till Google Chrome, Firefox, Safari eller Opera.

Att surfa är riktigt roligt men tyvärr kan det vara rätt plågsamt också, vissa sidor går extremt segt eller har alldeles för liten text. Det finns lösningar på detta, häng med så får du reda på mer!

  • För liten text?
  • Reklam som segar ner?

För liten text? Lätt fixat!

Ser du inte texten? För musen över texten för att förstora den, tänker inte fortsätta plåga dig utan berätta om hur du gör för att lätt fixa detta.

När texten är för liten kan du enkelt trycka på ctrl och + (på mac: cmd och +) för att förstora texten, tycker du texten är för stor kan du trycka på ctrl och - (mac: cmd och -) för att förminska den.

Trött på reklam som segar ner?

Ibland kan du märka att det går riktigt segt att besöka vissa webbplatser, särskilt de som kryllar av reklam. Förutom att sega när så kan det vara rätt trist att se all reklam. Ifall du använder Firefox eller Chrome som din webbläsare finns det enkla lösningar. Till Chrome kan du ladda ner tillägget AdBlock. Till Firefox finns också ett tillägg som heter AdBlock Plus som är oerhört populärt, i skrivande stund har tillägget laddats ner över 120 (!) miljoner gånger. Till Safari finns också möjligheten att få bort reklam genom AdBlock for Safari. För musen över bilderna nedan för att se skillnaden
